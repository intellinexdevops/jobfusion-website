# JobFusion Official Website

### 1. Clone this Project

```bash
git clone https://github.com/intellinexdevops/jobfusion.git
```

### 2. Run This Project

```bash
bun install

& bun dev
```

### 3. Lint

```bash
npx next lint

or

bunx next lint
```

### 4. Build

```bash
bun run build
or
bunx next build
```

### 5. Preview the production build

```bash
bunx next start
```
